Este é o repositório central da Documentação Técnica e Arquitetura de Software do Projeto Nex_TI, desenvolvido como pilar fundamental do Projeto Integrado Multidisciplinar (PIM III) da UNIP.

O Nex_TI redefine a experiência educacional ao fundir ciência cognitiva e tecnologia de ponta. Através da implementação do algoritmo de Repetição Espaçada (SM-2), sistemas dinâmicos de gamificação (XP e moedas virtuais) e um Agente de IA Especialista, o sistema garante que a retenção de conhecimento seja tão eficiente quanto engajadora.
